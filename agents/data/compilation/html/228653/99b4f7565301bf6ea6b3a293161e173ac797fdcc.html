UTMB's Artificial Intelligence

From UTMB's Artificial Intelligence Council: 

When using or consulting with generative AI, or similar technologies for clinical or administrative work, faculty and staff should be mindful of the security implications of information shared with AI tools. This especially includes disclosure of confidential or protected information. Importantly, entering such information into the public ChatGPT web interface is not in compliance with UTMB or UT System policies. For more information, please visit the pages below. 

  • UTMB Artificial Intelligence (AI) Center
  • UTMB Joint Policy on AI
    Scroll down to Acceptable Use of ChatGPT and Generative AI Tools section which includes information on Acceptable Use, Prohibited Use, and Use in Academic Work.

Approved HIPAA-Compliant UTMB Generative AI Tools

  • Chat Tools
  • Notetaking Tools
  • Ambient Dictation Tools
  • Co-Pilot for Microsoft 365
    Microsoft's generative chat experience available both as a chat app and embedded within applications across the Microsoft 365 portfolio (e.g. Excel, Word, Outlook). Access is available upon request by clicking the link above but note that access is not free and costs will be charged back to the User's Department.
    more... less...
    In your chat user interface, you may see a toggle switch between "Work" and "Web" (see below). If you are including patient or otherwise private data in your prompt, you must have "Work" selected and not "Web" as use of the "Web" feature (which allows the model to use internet resources to help answer your question) is not covered under HIPAA at present, we will provide updates here as this evolves.
  • Qualified Health
    UTMB's preferred tool for users looking for a generative Chat experience (akin to ChatGPT) with data that may contain protected health information or other private information. If you have a utmb.edu email address then you already have access at no additional charge. Use your UTMB SSO to log in to the resource.
  • UTMB Co-Pilot
    UTMB's internally-developed chat and co-pilot experience ideal for use cases such as administrative reporting, especially when wanting to work with data that exists within POWER. Access is available upon request by clicking the link here at no additional charge.
  • Microsoft Teams Premium
    UTMB's only endorsed tool for users wishing to leverage AI note taking during meetings. If you have access to this tool, you will see a Co-Pilot icon within your existing Microsoft Teams application.
    more... less...
    Access can be made available upon request by clicking the link above. The link will take you to a page to request access via Ivanti (note that your manager must be the one who submits the request). In the request form, select "Teams Premium" to specifically request CoPilot in Teams.
  • Nuance DAX
    UTMB's endorsed tool for clinical ambient dictation. This means using AI to transcribe the conversations during patient encounters into clinical notes and documentation. Access can be made available upon request by clicking the link above. The link will take you to a page to request access via Ivanti.
