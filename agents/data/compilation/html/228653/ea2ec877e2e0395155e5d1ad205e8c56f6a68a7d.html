HIPAA-Compliant Chat Tools

Qualified Health is UTMB's preferred tool for users looking for a generative Chat experience (akin to ChatGPT) with data that may contain protected health information or other private information. If you have a utmb.edu email address then you already have access at no additional charge. Click on the picture or link above or go to https://chat.qualifiedhealthai.com and log in with your existing UTMB single-sign-on (SSO) credentials.

 

 

UTMB Co-Pilot is UTMB's internally-developed chat and co-pilot experience ideal for use cases such as administrative reporting, especially when wanting to work with data that exists within POWER. Access is available upon request by clicking the link here at no additional charge.

 

 

Co-Pilot for Microsoft 365 (a.k.a. Microsoft Digital Experience) Microsoft's generative chat experience available both as a chat app and embedded within applications across the Microsoft 365 portfolio (e.g. Excel, Word, Outlook). Access is available upon request by clicking the link above or the logo but note that access is not free and costs will be charged back to the User's Department.

Important note about Co-Pilot for Microsoft 365: In your chat user interface, you may see a toggle switch between "Work" and "Web" (see below). If you are including patient or otherwise private data in your prompt, you must have "Work" selected and not "Web" as use of the "Web" feature (which allows the model to use internet resources to help answer your question) is not covered under HIPAA at present, we will provide updates here as this evolves.

It is also important to note that even when Work is selected as the toggle, you must still click on the three dots and toggle web search OFF in the menu in order to guarantee that web search is turned OFF to maintain HIPAA compliance.

 

 

HIPAA-Compliant AI Notetakers

Microsoft Teams Premium is UTMB's only endorsed tool for users wishing to leverage AI note taking during meetings. If you have access to this tool, you will see a Co-Pilot icon within your existing Microsoft Teams application:

Clicking this icon will open a chat experience within Teams where you can access several valuable features such as meeting summaries, action items, and much more:

Access can be made available upon request by clicking the link above or clicking the link here. The link will take you to a page to request access via Ivanti (note that your manager must be the one who submits the request). In the request form, select "Teams Premium" to specifically request CoPilot in Teams.

 

HIPAA-Compliant Ambient Dictation Tools

Nuance DAX (a.k.a. Microsoft DAX or "Dax Co-Pilot") is UTMB's endorsed tool for clinical ambient dictation. This means using AI to transcribe the conversations during patient encounters into clinical notes and documentation. Access can be made available upon request by clicking the link above or clicking the link here. The link will take you to a page to request access via Ivanti.
