Artificial Intelligence (AI)

Artificial Intelligence (AI) itself is identified as the capability of a computer program or machine to think, learn and take action, without being encoded with commands to complete those task. 

There are many tools that are AI-based used today. ChapGPT is one that generates new text or code based on a query. Grammarly gives writing or paraphrasing suggestions. DALL-E will generate unique images based on text prompts and Synthesia will create spoken language or talking video avatars based on a text prompt. 

When using any of these AI based tools, make sure to follow the AI Information Literacy guidelines to help ensure the ethical use of these tools. 

Generative AI

Generative AIs, such as ChatGPT, are technologies that can create content, such as text, images, code, audio and video, based on the prompt provided and information accessible by the AI. These results are based on patterns and probabilities that the AI was programed to interrupt. 

ChatGPT was developed by a company called OpenAI. The GPT in ChatGPT stands for generative pretrained transformer. Pretrained refers to the data it was trained on to learn patterns and relationships between words and phrases. Transformer is the part of the machine learning model that helps understand the context of the words.

It is a large language model AI, which is a tool that can generate human-like text based on prompt context, training data utilized and past interactions. It is based on training data found on the internet from pre-2021.

ChaGPT can assist with writing tasks, such as brainstorming ideas, composing an outline and generating research questions. Shortcomings of the tool is that the results will be generated from information from 2021 and before, it is providing a text overview of the information searching, can be biased, as this is integrated into the training materials, and each prompt will generate different results. 

 

AI Concerns

One way to make sure the AI generated materials are used properly is to follow the AI Information Literacy guidelines provided in this guide. These guidelines provide the information needed to critically analyze and ethically use these AI tools.

Other things to be aware of is the accuracy of the information provided. Generative AI is intended to provide a text base response based on the query provided and the information searched. And it will produce those results, whether it the information it is searching is accurate or not or whether it completely understands the full context of the located results. 

Because generative AI is providing a text based response, there is no list of sources for the information provided. There is no way to give credit to who created the information and there is no way to verify the accuracy of it. 

Additionally, AI uses a lot of energy to train in the data and language and to generate the results. These processes have put a major drain on the energy in the business' area.  

AI Uses

There are many uses for AI and more specifically generative AI. Some of these include:

  • Brainstorming-Generative AI can help organize thoughts, provide associated subject terms, develop an outline, etc. 
  • Understanding-AI can be used to help break down large concepts or ideas for easier understanding
  • Translating-AI tools can be used to translate languages
  • Code-It can be used to write code or clean up existing code
