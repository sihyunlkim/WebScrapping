AI Tools

The list of tools below is not exhaustive, it is a starting point for faculty to explore. If you would like assistance on how to incorporate any of these tools into your course curriculum, course content or to increase student engagement, please reach out for a 1-1 consultation!

  • Anthropic Claude
    A competing popular general AI chatbot. It will create text-related content, images, and code.

    Conversational and Generative AI

  • Canva
    A popular free video and image editing tool. It features several generative AI functionalities, such as being able to create images through text, expanding backgrounds, moving photo elements, completely altering items within images, and more.

    Content Creation and Design

  • Canvas Discussion Summary
    Discussion Summaries use AI to automatically capture the key points, questions, and ideas from course discussion threads. Administrators or instructors can enable this feature to provide quick, instructor-only overviews of class conversations.

    Educational and Collaborative Tools

  • Canvas Smart Search
    Smart Search is an AI-powered Canvas feature that understands the context of user queries to deliver more accurate, relevant results without relying on traditional keyword searches. It helps users quickly find course content—such as pages, announcements, discussions, and assignments.

    Educational and Collaborative Tools

  • ChatGPT
    One of the most popular general AI chatbots. It will create text-related content, images, and code via chat prompts. It’s free to use, but it has some limitations without a paid account.

    Conversational and Generative AI

  • Consensus
    An academic search engine for research papers powered by generative AI. Users can search by their research questions, synthesize search results, and get a top-level overview of the search details.

    Research and Academic Support

  • Descript
    An AI video editor. The tool allows you to delete entire segments from your footage by editing text script. It will also allow you to create podcasts, clips, and more.

    Content Creation and Design

  • D-ID
    A generative AI tool that allows you to create custom digital avatars in photos and videos.

    AI for Personalization and Storytelling

  • Elicit
    An academic search engine for research papers powered by generative AI.

    Research and Academic Support

  • Gamma
    A tool that generates images, presentations, and slide decks through generative AI. It has some free options but requires a subscription for its best features.

    Content Creation and Design

  • Google Gemini
    Google's general generative AI chatbot. It will create images, text, and code.

    Conversational and Generative AI

  • Google's NotebookLM
    A personalized AI research assistant. One of its most interesting features is that it turns your notes into podcasts.

    Research and Academic Support

  • Grammarly
    A grammar checker and writing assistant powered by generative AI. Grammarly can be used to correct simple grammatical mistakes, but it also has the option to write entire documents.

    Writing and Language Tools

  • Grok
    Grok is an advanced AI assistant that delivers fast, intelligent answers with real-time internet-aware insights. It supports coding, content creation, image generation, and problem-solving with a witty, conversational tone.

    Conversational and Generative AI

  • Harmonize
    A discussion and collaboration tool with the look and feel of social media and a rich multimedia content editor. It can use generative AI to create prompts and even grade student submissions. It is integrated into Canvas.

    Writing and Language Tools

  • Ignite Canvas
    IgniteAI is a set of AI tools built into Canvas that streamlines grading, assessment creation, feedback, translation, and analytics. It helps educators and students work more efficiently while staying secure and fully integrated within the Canvas platform.

    Educational and Collaborative Tools

  • Kahoot!
    A game-based quizzing tool. Kahoot! now allows users to generate quizzes based on topics, PDF files, webpages, and more. Users can also generate questions based on topics or PDFs. All these features are powered by generative AI.

    Educational and Collaborative Tools

  • Kritik
    A peer-to-peer evaluation platform with Canvas integration. The tool reviews student comments and provides students with feedback using generative AI.

    Educational and Collaborative Tools

  • Meta AI
    Meta AI is a versatile assistant that provides fast, intelligent answers with fully integrated real-time information. It supports image generation, writing, coding, and everyday problem-solving with a natural, conversational tone.

    Conversational and Generative AI

  • Microsoft's Copilot
    Microsoft’s general AI chatbot, similar to ChatGPT. FIU faculty, staff, and students have free access to it through their email credentials on the Copilot website. Office app integration (Word, Excel, PowerPoint, and Teams) is currently unavailable at FIU. The tool will create text-related content, images, and code via chat prompts.

    Educational and Collaborative Tools

  • Perplexity
    An AI-powered answer engine that provides more accurate, trusted, and real-time answers to any question. The tool has a pro option that asks clarifying questions and refines queries.

    Conversational and Generative AI

  • Perusall
    A social annotation and collaboration tool. It allows students to add notes and comments to documents, web pages, and more. The tool uses generative AI to grade the quality of student submissions. Grades are automatically pushed to the Canvas grade book.

    Educational and Collaborative Tools

  • Research Rabbit
    Described as Spotify for researchers, this generative AI tool aims to create a more robust user experience by visualizing search results, making recommendations and suggestions for related topic areas, and updating the user on the latest papers related to their past searches.

    Research and Academic Support

  • Runway
    A generative AI tool for creating images, animations, and videos. Its impressive set of features will allow you to do anything from adding motion to still images to applying your facial expressions to performing CGI characters.

    Content Creation and Design

  • Suno AI
    A generative AI tool that allows you to create a song about anything.

    Content Creation and Design

  • Whimsical
    A generative AI tool that allows you to create flowcharts, sequence diagrams, and mind maps in seconds.
