OpenAI ChatGPT Edu

OpenAI ChatGPT Edu

Tools & services
Get started arrow_circle_right Request access arrow_circle_right

Overview

OpenAI ChatGPT Edu is a powerful chatbot tool including feature such as custom GPTs, Deep Research, web search, and canvas mode. Unlike the free version of  ChatGPT, it has enhanced security and privacy protections making it appropriate for use with up to Level 3 confidential data. This tool is designed for producing a wide array of outputs including text, images, and executable code from inputs that include text, files, or images. 

Compliance with University guidelines for use of generative AI required for use.

Eligibility

Faculty, staff, and researchers in FAS, College, GSAS, SEAS, GSD, GSE, HDS, Radcliffe, SPH; Students in College

Access approved and coordinated by Schools.

Data classification level

Approved for up to Level 3 confidential data 

Data you enter will not be used to train OpenAI’s models
