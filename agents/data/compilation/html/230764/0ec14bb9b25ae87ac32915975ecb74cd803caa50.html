ChatGPT at The U

  • Get Started with ChatGPT
  • Utilizing ChatGPT
  • Additional Resources

ChatGPT Edu provides the university community with secure, institutionally supported access to advanced AI capabilities within a university-managed environment designed to support teaching, learning, research, and administrative work.

As the university evaluates and deploys AI-powered tools, protecting institutional data remains a top priority. ChatGPT Edu does not use university data for model training, and campus-wide security controls make it a safer option than personal accounts.

What's Included in ChatGPT Edu?

  • Canvas
  • Codex
  • Web search
  • Deep research
  • Image generation
  • Voice
  • Models
    • GPT-5.2 Thinking (thinks longer for better answers)
    • GPT-5.2 Pro (research-grade intelligence)
  • GPTs
  • Projects

 

Looking Ahead

University leaders are analyzing usage data to understand community needs and shape a long-term, cost‑effective service model that retains as many capabilities as possible.

 

AI models

ChatGPT Edu provides access to a set of OpenAI models and features designed to support academic, research, and operational use, including tools for advanced data analysis, multimodal input, and search.

Knowledge

ChatGPT Edu draws on a broad, general body of world knowledge and can reference real-teim information through search, but does not learn from or store individual user conversations.

Data Protection

No university data is used to train OpenAI models, and all ChatGPT Edu activity operates within enhanced privacy and security controls aligned with university data-governance requirements.

Sources

When using search, ChatGPT Edu can pull from current, publicly available information and provide source citations, enabling users to verify and follow up on its responses.
