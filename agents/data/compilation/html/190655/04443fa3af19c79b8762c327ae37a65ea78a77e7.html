AI Resources & Tools

Microsoft Copilot

Microsoft Copilot is a generative AI assistant integrated into Microsoft 365 apps like Word, Excel, Outlook, PowerPoint, and Teams. It helps users write, summarize, analyze, visualize, and automate tasks using natural language prompts-similar to tools like ChatGPT or Google Gemini.

CoPilot is available to all City Tech faculty and staff through the Microsoft 365 platform. You can use Copilot with your data in a more secured way.

The version of Copilot currently available does not integrate with institutional data such as emails, documents, calendars, or SharePoint files. That functionality requires a higher-tier license (Microsoft 365 Copilot for Enterprise), which is not yet available at our college.

App Copilot Capabilities
Word Drafts documents, rewrites text, summarizes content
Excel Analyzes data, creates formulas, generates charts
PowerPoint Builds presentations from outlines or documents
Outlook Summarizes threads, drafts replies, prioritizes emails
Teams Summarizes meetings, suggests action items, answers questions from chat history

How to Use Microsoft Copilot

  1. Look for the Copilot icon in your Microsoft 365 apps.
  2. Use natural language prompts like:
    • "Summarize this document"
    • "Create a chart from this data"
    • "Draft an email to follow up on this meeting"
  3. Refine the output by asking Copilot to adjust tone, length, or format.

Best Practices for Using Copilot

  • Start simple: Use Copilot to automate repetitive tasks like summarizing notes or formatting slides.
  • Review everything: Always verify Copilot's output for accuracy and tone.
  • Protect sensitive data: Avoid prompting Copilot with confidential or personally identifiable information.
  • Collaborate with Copilot: Think of it as a creative partner, not a replacement for your judgment.

Zoom AI

Zoom AI Companion is a generative AI assistant built into Zoom that helps users work more efficiently by summarizing meetings, drafting messages, generating ideas, and automating routine tasks. It functions similarly to tools like ChatGPT or Google Gemini, but is embedded directly into the Zoom platform.

The version currently available integrates with institutional data on a limited basis.

Feature Description
Meeting Summaries Automatically generates summaries of Zoom meetings with key points and action items.
Smart Replies Suggests responses in Zoom Team Chat based on context.
Email Drafts Helps draft follow-up emails after meetings.
Idea Generation Assists with brainstorming and content creation.
Task Automation Identifies and tracks action items from conversations.
