    AI - Introduction to the TAMUS AI Chat Framework

    Tags AI

    Overview

    The Texas A&M University System AI Chat (TAMUS AI Chat), currently in BETA, provides staff, faculty, researchers, and students with a secure, university-approved platform to access multiple artificial intelligence (AI) tools like OpenAI’s GPT, Anthropic’s Claude Sonnet, and Google’s Gemini. The following https://docs.tamus.ai are available:
     NOTE: Claude Sonnet 3.5 has been retired—please use Claude Sonnet 3.7 or 4. [Read more on the blog]

    • Getting Started Guide: A comprehensive introduction to using the features and tools within the AI Chat interface.
    • FAQs: A collection of frequently asked questions, support contact details, and a search function to find resources related to the AI Chat Preview.
    • Model Descriptions: Detailed information on various AI models, including their costs, capabilities, and optimal use cases.
    • API Documentation: Detailed technical information for developers on integrating AI Chat capabilities with other systems.
    • Blog: Stay informed about updates to TAMUS AI Chat, including model additions or retirements, feature changes, and other important announcements.
