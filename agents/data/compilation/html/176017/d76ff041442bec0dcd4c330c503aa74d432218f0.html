Introduction to Google Colab: Using Generative AI for Python Programming In-Person

This is an introductory workshop on Google Colab and how to use it for Python programming.

In this workshop, you will learn:

  • Basic features of Google Colab for Python programming
  • A demonstration of key features
  • How to use Colab's Generative AI tools for Exploratory Data Analysis (EDA)

To fully participate in this hands-on workshop, please bring your own laptop with internet access. There is no need to install anything, as Google Colab runs in the cloud, but having a Google account is recommended.

Please note that this workshop will NOT teach you how to write Python code. Instead, it will focus on how to use Google Colab for Python programming. You can still attend without prior Python knowledge, but it will be beneficial if you have basic knowledge of Python.

For additional information, contact Dr. Sian Lee at slee63@olemiss.edu.
