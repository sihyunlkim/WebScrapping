Guidance Using Student Data with ChatGPT

NC State employees have a responsibility to protect university data when using AI software

July 10, 2023 Madalene Adams 1-min. read

As we learn more about the abilities of ChatGPT and other AI software you may be curious how this new technology can support your work. Please be advised that NC States Office of Information Technology has recently published guidance on using such tools with university data. Per this guidance, you can only submit green data (non-sensitive data) into AI tools. Much of the data used by DASA units (including demographic data) is not considered green data and cannot be shared with these tools. Be aware that ChatGPT is not FERPA compliant.

All NC State employees have a responsibility to protect university data and to adhere to the Data Management Framework. Please ensure that you review this guidance and understand how to keep data safe. If you have any questions about data management, please reach out to DASA Assessment or DASA Tech staff.
