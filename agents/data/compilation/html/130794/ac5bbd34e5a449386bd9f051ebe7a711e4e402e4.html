Clarity Platform

What is Clarity

Clarity is a secure, scalable AI platform that provides Yale faculty, students, and staff access to different AI models (like GPT, Claude, etc.) to support their work and research. Clarity offers enterprise-level security, assuring users that Yale data is safe and that conversations will not be used to train AI models, while providing the flexibility to select the best AI tool for a specific task.

  • Secure Connections: Instead of connecting to AI services directly (which can be less secure), Clarity uses private, encrypted connections with proper security credentials. It’s like having a private, guarded tunnel instead of using a public road. 
  • Multiple Ways to Use AI: You can interact with AI via the chat interface at clarity.yale.edu, or other programs can connect via the API to enable AI features programmatically.
  • Detailed Permission System: We control exactly who can do what on the platform. For example, some people might be allowed to create AI agents, while others might only be able to chat with them, and others will be able manage data. It’s like having different key cards that unlock different doors in a building.
  • Choice of AI Models: Users can use different AI models for different tasks - like using one model for understanding questions and another for generating answers.
  • Smart Data Integration: Clarity can pull in relevant information from your files and other data sources to give AI models better context for more accurate answers. Clarity also employs advanced techniques that are faster and more cost-effective than basic methods.
  • Consistent Experience: Clarity provides a single interface, so users get a consistent experience regardless of which AI model they’re using.
  • Built for Scale: Clarity can handle thousands of users across the entire university without slowing down.
  • Accessible Design: Our websites follow accessibility guidelines (WCAG 2.1 AA), so they work well for users with disabilities.
  • Robust Infrastructure: Clarity runs on Yale’s secure cloud systems, which manage everything from user logins to running code safely and managing connections to AI services.
clarity.yale.edu

Security

Clarity is appropriate for use with low, medium and most high-risk data, with the notable exception of Electronic Protected Health Information (ePHI). Users with external obligations like data use agreements should also make sure their use of Clarity is appropriate. For further details, see Clarity Platform Security.

How Do I Get Support?

​If you are experiencing an issue with your use of the Clarity platform that is not addressed by the content on this website, please reach out in the following ways.  

  • Review the known issues to determine if your problem has already been reported.
  • ​For technical support related to the Clarity platform please reach out to the ITS helpdesk.
  • ​For support related to AI in teaching and learning, please see the AI Guidance for Teachers or reach out to askpoorvucenter@yale.edu.
