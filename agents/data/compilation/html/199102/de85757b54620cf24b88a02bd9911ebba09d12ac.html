1. Developing keywords & search strings

For this step, you can use generative AI tools, which will be listed below. Try the following prompt:

Act as an expert academic librarian. My research topic is [chosen topic]. Could you please give me a search string I can use to search for that topic in library databases? And tell me which databases I should use.

 

  • ChatGPT
    Text-based generative AI tool created by OpenAI
  • Google Gemini (fka Bard)
    Text-based generative AI tool created by Google
  • Bing Copilot
    Text-based generative AI tool created by Bing/Microsoft

A note on prompt generation

Using a detailed prompt is essential when using AI chatbots. Consider using the formula below to create prompts so that you get the output you need:

  • Role: Define the role you want the AI to assume
  • Context: Provide additional context for depth and relevance
  • Action: Specify the task at hand
  • Expectations: Give clear instructions for what needs to be accomplished

2. Searching the literature

In addition to library databases, you should consider searching for your topic in AI literature search tools (below), which have some unique capabilities.

  • Semantic Scholar
    Literature search tool that simplifies search results and points out influential citations
  • Elicit
    Literature search tool that provides one sentence abstract summaries and extracts details from papers into a table
  • Inciteful
    AI literature search tool that builds a network of papers from citations and gives you the information you need to quickly get up to speed on a topic.
  • Litmaps
    AI literature search tool that uses citation networks to discover literature and allows you to set alerts for new papers

3. Reading sources

The tools below work like chatbots. Once you upload the PDF of an article, you can ask the tool questions about it.

  • Humata
    AI chatbot that can digest papers and then summarize and answer questions about them
  • ChatPDF
    AI chatbot that can digest papers and then summarize and answer questions about them

4. Writing your manuscript

For this step, you can use generative AI tools, which will be listed below. Try the following prompt:

Act as an expert academic editor. Provide feedback on the submitted writing, highlighting areas for improvement and suggesting ways to enhance the piece's overall quality: [Provide your writing sample]

 

  • ChatGPT
    Text-based generative AI tool created by OpenAI
  • Google Gemini (fka Bard)
    Text-based generative AI tool created by Google
  • Bing Copilot
    Text-based generative AI tool created by Bing/Microsoft

Other notable AI tools for research

Below are other AI tools by category that may aid in the research process.
 

Function Tools
Data analysis tools Tableau - Data analytics and visualization platform; Tableau Pulse and Einstein Copilot are AI options within the platform (requires paid subscription to access)
Systematic/scoping review tools

Covidence - Tool for screening and extracting data from studies for reviews; uses machine learning to predict studies' relevancy (tool paid for by N.C. A&T; sign up for an account)

ASReview - Tool for screening and extracting data from studies for reviews; uses machine learning to predict studies' relevancy
Transcription tools

Zoom AI - Summarizes and transcribes meetings and chats (tool paid for by N.C. A&T)

Notta - Summarizes and transcribes audio/video recordings (requires paid subscription to access)
Textual analysis tools Voyant - Web-based reading and analysis environment for digital texts
