AI, ChatGPT, and the Library: Learn More about Generative AI

This guide provides a starting place for MCC faculty, staff, and students to learn how generative AI intersects with information literacy and how to use it responsibly.

Learn More

Is generative AI a new concept for you? Have you dabbled with ChatGPT and want to learn more? Browse this reading list by category (what it is, education, issues). Select readings using the hyperlinked titles.

MCC Faculty and Staff, visit the MCC Center for Teaching and Learning guide on AI to learn more, find resources, and engage in discussions with colleagues about generative AI in higher education.
