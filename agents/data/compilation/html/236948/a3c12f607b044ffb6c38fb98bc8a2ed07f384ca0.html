Need help with Canvas?

The Teaching@UW website has information on pedagogies that support digital learning. However, if you need assistance learning how to use the Canvas learning management system (e.g., creating quizzes, building pages, grading with Speedgrader, etc.), please contact the support team at help@uw.edu.
