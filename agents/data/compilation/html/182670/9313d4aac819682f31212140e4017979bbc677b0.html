AI Tools

If you plan on providing data to LLMs, see our Dartmouth Generative AI Tools — Data Sensitivity Guide

Dartmouth-Hosted Open Models and Tools (Free):

  • Llama, Mistral, Gemma: Meta, Google, etc. opens large language model. Instruction and foundation models. This is also an image-based multimodal model. All data remains local to Dartmouth. Access via Dartmouth Chat. Direct access to detailed documentation of Dartmouth Chat.

Commercial Tools:

  • Campus-wide licensing:
    • OpenAI's GPT models : free access for all Dartmouth users through chat.dartmouth.edu. Data is sent to OpenAI and may be used for future model training.
    • Anthropic's Claude: free access for all Dartmouth users through chat.dartmouth.edu. Data is sent to Anthropic.
    • Microsoft Copilot chat bot: free access for all Dartmouth users through our enterprise license with Microsoft. Based on GPT 4. Offers data protection for any input. Includes access to DALL-E3 image generator.
  • Freemium/subscription licensing:
    • OpenAI’s ChatGPT: limited free access. Ability to upload and analyze files, access DALL·E image generator, and create custom GPTs.
    • Anthropic’s Claude: unusually large text input window. “Pro” subscription gives access to stronger models.
    • Perplexity AI:  particularly strong in returning citations and sources on which answers are based, research-assistant like. “Pro” subscription gives access to a variety of LLMs (GPT 4, strong versions of Claude). Can ingest and analyze files.
    • Elicit: great tool far AI-assisted academic literature searches. Free trial, then a “Plus” subscription is needed.
    • GitHub copilot: subscription-based tool to assist with coding.
    • Google Gemini: Google’s AI tool, “Gemini Advanced” subscription offers a stronger model and integration with Gmail and Google docs.
    • Aggregator site: https://theresanaiforthat.com/

Other Dartmouth-hosted tools:

  • Dartmouth hosts a number of customized AI tools at ai-tools.dartmouth.edu. Offers increased protection for data input. No copy of the data is saved after the session expires.
