When to Cite

For Chicago, it is recommended that you cite ChatGPT for any generated text which you have used (whether paraphrased or directly quoted). For other uses of the software, you should acknowledge the software in the text of your document (e.g. "ChatGPT was used to compile a list of Regency era poets.")

You should

  • cite a generative AI tool whenever you paraphrase, quote, or incorporate into your own work any content (whether text, image, data, or other) that was created by it 
  • acknowledge all functional uses of the tool (like editing your prose or translating words) in a note, your text, or another suitable location 
  • take care to vet the secondary sources it cites

Examples

Footnote version:

To cite text generated by ChatGPT, they offer the following example:

1. Text generated by ChatGPT, March 7, 2023, OpenAI, https://chat.openai.com/chat.

Do not include an entry in the reference list, as the infomation is not retrievable.

Author-Date version:

Treat in a similar manner to personal communication:

(ChatGPT, March 7, 2023).

Once again, do not include an entry in the rference list.
