AI Smart Pass Program

The Office of Information Technology (OIT) invites Georgia Tech faculty and staff to participate in the AI Smart Pass loaner program, which provides temporary access to Copilot for Microsoft 365 and Open AI ChatGPT Edu. This program is for short-term, project-based use across academic, research, and administrative units, allowing users to explore AI-driven innovation.

Get Started >

Personal Assistant | AI Chatbot Tools

GPT-powered, conversational and generative AI for text, code and image generation.

The Copilot chat tool offers text, graphic, and code generation with commercial data protection. The tool delivers the same GPT 4.0 capabilities as other popular genAI tools like ChatGPT without leveraging Georgia Tech data to train its model. Copilot also provides copyright protection and ensures that any content generated using the tool is the intellectual property of Georgia Tech.
