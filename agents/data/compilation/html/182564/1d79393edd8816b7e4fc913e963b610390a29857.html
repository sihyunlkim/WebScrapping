Welcome to the Niche Academy Tutorials Guide!

In this guide you will find information and links to the various tutorials available through Niche Academy, a learning platform that has been designed for use in libraries and educational centers.

Please note that many of these tutorials have been created by other colleges and universities and may refer to services and contact information for those institutions. For questions or assistance, please contact the librarians at WNC at refdesk@wnc.edu, rather than using the tutorial contact links.

To help direct your search for information, we have organized the tutorials into sections.

In the left-hand navigation tabs, you will find:

  • Database Tutorials - In this section you will find tutorials on the various databases that WNC subscribes to, as well as open access options.
  • Research Skills - These tutorials focus on tips and tricks to use during the research process.
  • Software Skills - Included in this section are tutorials on various programs and software, including Zoom, Microsoft, and Google.
  • Other Useful Skills - The tutorials in this section pertain to miscellaneous topics, including how to use a Chromebook, password creation, online job application tips, Zoom, and more.
  • Instructional Resources - Use these tutorials to learn more about creating web-based lectures, OERs, engaging with online learners, and other useful information.
  • Professional Skills - This section focuses on various skills that are particularly useful to professionals that work with a variety of communities and individuals.
  • All Tutorials - This section contains all of the Niche Academy Tutorials selected by WNC.

If you have any questions or need assistance, please contact us and the library staff will be happy to help.
