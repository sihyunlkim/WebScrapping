AI @ Washington State University

Artificial intelligence (AI) is a rapidly evolving field that is transforming the way we live, work, and interact with the world around us. From self-driving cars to virtual assistants to medical diagnosis, AI is having a profound impact on our society.

This guide is designed to help the WSU community learn more about AI within writing, research, and teaching. We provide resources for incorporating AI within the research process, WSU specific information, citations, teaching, and publishing.

Whether you are a student, teacher, researcher, or a simply curious about AI, we hope that this guide will be a valuable resource for you.

What is AI?

Artificial intelligence (AI) is the ability of a machine to simulate human intelligence. This includes tasks such as learning, reasoning, and problem-solving.

Generative AI (ChatGPT, Google Bard, Bing AI) is a type of artificial intelligence (AI) that can generate new content, such as text, images, or music. It does this by learning from existing data and then using that data to create new outputs.

Popular Generative AI Tools

  • Microsoft Copilot
    Copilot is the new name of Bing Chat, the AI-based search function of the Bing web browser. Copilot is powered by the same GPT-4 technology used by ChatGPT. It can generate text, translate languages, write different kinds of creative content, and answer your questions in an informative way. Under the universityâ€™s current Microsoft licensing agreement, Microsoft Copilot is now available to all currently enrolled students and active employees with a WSU Network ID (NID).
  • ChatGPT
    A generative AI tool from OpenAI that can generate text. It can be used to generate different creative text formats of text content, like poems, code, scripts, musical pieces, email, letters, etc. It can also be used to translate languages and write different kinds of creative content.
  • Google Gemini
    An experimental large language model (LLM) chatbot developed by Google AI, trained on a massive dataset of text and code. Google Gemini (formerly Google Bard) can generate text, translate languages, write different kinds of creative content, and answer your questions in an informative way.
  • Claude
    Claude is a generative AI tool developed by Anthropic. The free tier of Claude allows multiple uses, including chatting, creating written content, generating code, visualizing data, and analyzing text and images.
