To access tools with the protections of UC-Berkeley licensing, users must authenticate through CalNet and, in some cases, request a Berkeley-provided license. (See the "Access" row in each table below.) Users are reminded to consult the campus guidance on the Appropriate Use of Generative AI Tools.
